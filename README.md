# FitBit Health & Activity Analytics Dashboard ⌚📊

## 📌 Overview

This project presents an interactive Power BI dashboard designed to analyze Fitbit health and physical activity data.

The dashboard focuses on daily activity monitoring, calorie tracking, movement analysis, and behavioral health insights.

---

## 🚨 Business Problem

Fitness tracking devices produce detailed activity data every day.

However, raw data alone does not clearly answer questions such as:

- How active is the user daily?
- What is the average movement behavior?
- How do activity intensity levels differ?
- How much sedentary behavior exists?
- How do activity patterns change over time?

This dashboard transforms wearable health data into actionable insights.

---

## 🎯 Objectives

- Monitor daily health activity.
- Analyze calories burned.
- Track steps and traveled distance.
- Evaluate activity intensity levels.
- Compare yearly activity patterns.
- Study sedentary behavior.

---

## 🛠 Tools Used

- Power BI
- DAX
- Data Modeling
- CSV Dataset

---

## 📊 Key KPIs

| KPI | Description |
|------|------|
| Calories | Daily calorie consumption |
| Steps | Daily walking performance |
| Distance | Traveled distance |
| Floors | Floors climbed |
| Activity Minutes | Movement duration |
| Sitting Minutes | Sedentary behavior |

## 📈 Dashboard Features

### Activity Monitoring

- Daily steps analysis
- Calories tracking
- Distance monitoring
- Floors analysis

### Activity Behavior Analysis

- Slow activity analysis
- Moderate activity analysis
- Intense activity analysis
- Daily movement trends

### Time Analysis

- Monthly filtering
- Day-level activity tracking
- Year comparison analysis (2015–2016)

---

## 🔍 Key Insights

- Average daily steps reached approximately **11.7K**.
- Daily calories averaged around **3K calories**.
- Activity behavior varies between slow, moderate, and intense movement levels.
- Daily activity trends fluctuate across the month.
- Sedentary tracking provides insights into inactive periods.

This dashboard helps users:

- Monitor physical activity

- Track fitness behavior
- Identify inactivity patterns
- Support healthier daily decisions
